# BurnBite

Kişisel beslenme, makro ve kilo takip uygulaması.

## Özellikler

- Günlük kalori, protein, yağ ve karbonhidrat takibi
- Hazır Türk mutfağı menüsü + kendi yemek ekleme
- Su ve egzersiz takibi
- Haftalık kilo kaydı ve geçmiş görünümü
- Mobil-first PWA — telefonda ana ekrana eklenebilir
- Veriler tarayıcıda (localStorage) saklanır

## Geliştirme

```bash
npm install
npm run dev
```

## Deploy

Vercel ile otomatik deploy edilir.
